ממשק המעצב (לקוח)

1. בד וצבע

גלילה אופקית בין 44 גוונים. לחיצה על צבע מעדכנת את השקית בזמן אמת. אפשר להעתיק SKU/HEX.

2. לוגו

העלאת קובץ → גרירה על השקית או סליידרים (גודל, מיקום, סיבוב). צילום PNG מהסרגל הימני.

3. הצעת מחיר

בחירת גודל שקית (p1–p14), כמות, משלוח, ידיות, למינציה, צבעי לוגו. המחיר מתעדכן אוטומטית מהמחירון.

מילוי שם / אימייל / טלפון → הורד PDF — כולל צילום 3D וסיכום ₪.

זרימת CRM (Eli)

Dashboard → פתח ליד → כפתור "מעצב 3D" → WhatsApp CTA ללקוח → לקוח פותח קישור אישי (?t=...) → עיצוב + PDF → עיצוב נשמר בליד + הערה + לוג פעילות

הכפתור נמצא בסקירה ובצ׳אט (Composer). בלשונית סקירה מופיעים עיצובים שמורים.

רשימת בדיקה (אחת־אחת)

  1. פתח http://localhost:3000/configurator — וודא שהשקית נטענת
  2. בחר צבע — השקית משתנה
  3. העלה לוגו — מופיע על השקית
  4. לשונית הצעת מחיר — שנה כמות — המחיר מתעדכן
  5. הורד PDF — קובץ עם מוקאפ ומחיר ₪
  6. פתח http://localhost:8081/configurator — iframe לאותו מעצב
  7. Dashboard → ליד → מעצב 3D (דורש DB + Bridge)
