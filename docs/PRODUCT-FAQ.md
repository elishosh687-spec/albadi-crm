# Albadi — Product FAQ (LLM context)

> נטען אוטומטית לתוך כל קריאת LLM של ה-autoresponder (spec-extractor + unmatch-agent).
> עדכן כאן כשמשתנה מידע מוצר — ה-bot ייקח את העדכון ב-deploy הבא.
> אורך מטרה: ≤ ~600 tokens. אל תכתוב כאן ספר — רק עובדות יבשות שה-LLM צריך כדי לענות.

## חומר

- חומר ראשי: **PP non-woven** (פלסטיק נארג, "פולי-פרופילן")
- משקלים: 80gsm רגיל, 100gsm חזק יותר
- עמיד, שטיף, ניתן למחזר, ידידותי לסביבה יותר משקיות פלסטיק חד-פעמיות
- שימושים: שקיות שיווק, אירועים, חנויות, מתנות לקוחות

## הדפסה

- שיטה: **סילק סקרין** (silk screen)
- צבעים בודדים: עד 4 צבעי דפוס נפרדים
- "פול קולור" / CMYK / הדפסה דיגיטלית — אופציה, לפי בקשה (תמחור ידני)
- מינימום שטח לוגו: כ-3×3 ס"מ
- אזורי הדפסה: צד אחד או שני צדדים (משפיע על מחיר)

## כמויות

- **מינימום: 500 יחידות.** מתחת ל-500 — לא מייצרים.
- כמויות סטנדרטיות בשאלון: 1,000 / 3,000 / 5,000 / 10,000
- כל כמות אחרת = "אחר" → תמחור ידני של אלי (factory route)

## מידות סטנדרטיות

- 20×8×25 ס"מ — קוסמטיקה, תכשיטים
- 30×10×30 ס"מ — ביגוד קל, מתנות
- 40×12×30 ס"מ — נעליים, ביגוד
- 40×15×50 ס"מ — פריטים גדולים
- 30×40 ס"מ — שטוחה, פריטים רחבים
- 20×15 ס"מ — קטנה, פריטים קטנים
- כל מידה אחרת = "מידה מיוחדת" → תמחור ידני

## זמני אספקה

- **אקספרס: ~25 יום** מאישור הזמנה
- **רגיל: ~90 יום** מאישור הזמנה
- אספקה כוללת: ייצור + הדפסה + משלוח לארץ + מע"מ
- הבוט לעולם **לא** מאשר תאריך ספציפי — רק את ה-tier (אקספרס / רגיל)

## תשלום

- **50%** בעת אישור ההזמנה (מקדמה)
- **50%** לפני יציאת הסחורה מהמפעל
- מטבע: ILS (כל המחירים)

## כלול במחיר

- ייצור השקיות
- הדפסה (לוגו / טקסט)
- גלופה (הכנה לדפוס)
- משלוח לארץ
- מע"מ

אין עלויות נסתרות.

## ידיות

- אופציה: **עם ידיות** (חבל / רצועות / קרטון) או **בלי**
- עם ידיות = מעט יקר יותר
- סוג ידיות ספציפי / מספר ידיות = אלי מתאם אישית

## לוגו / קבצים

- פורמטים מקובלים: **PDF, JPG, PNG, SVG, AI, EPS** — כל פורמט בסדר
- וקטור (SVG / AI / EPS) עדיף איכותית, אבל לא חובה
- "אין לי לוגו" → אלי מתאם שירות עיצוב ידני בתשלום נפרד
- הבוט **לא** מנתח תוכן הלוגו — מקבל, מעביר לאלי

## אחריות + איכות

- בדיקת איכות לפני שליחה מהמפעל
- 14 יום בדיקת מוצר אצל הלקוח אחרי אספקה
- בעיות איכות → החלפה / זיכוי

## משלוח

- כל מחיר כולל משלוח לארץ
- אזורי משלוח: כל הארץ (כולל אילת + אזורים מרוחקים)
- חבילות גדולות מאוד → תיאום ידני

## מה הבוט **אסור** לעשות

1. **לעולם לא לצטט מחיר** — כל מחיר בא מהמחשבון (`bag-quote-app`) או מאלי.
2. **לעולם לא לאשר תאריך אספקה ספציפי** — רק tier ("אקספרס" / "רגיל").
3. **לא להבטיח זמינות / production slots.**
4. אם הלקוח מבקש אדם / טלפון → escalate לאלי.
