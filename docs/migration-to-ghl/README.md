# docs/migration-to-ghl/

כל המסמכים הקשורים למעבר מ-dashboard v3 ל-GoHighLevel.

## קבצים

| קובץ | תפקיד |
|---|---|
| [PLAN.md](PLAN.md) | תוכנית מלאה — phases, status, IDs |
| [HANDOFF.md](HANDOFF.md) | snapshot לסשן חדש — מה עשינו, מה נשאר, IDs |
| [TESTS.md](TESTS.md) | בדיקות ידניות per phase |
| [DELETION-CHECKLIST.md](DELETION-CHECKLIST.md) | מה למחוק מ-albadi-crm אחרי שmigration יציב |
| [ARCHITECTURE.md](ARCHITECTURE.md) | איך הקוד יחיה אחרי שdashboard נמחק |

## אם אתה בסשן חדש

1. קרא `HANDOFF.md` — תקבל snapshot מלא
2. אחר כך `PLAN.md` — phases + סטטוס
3. אם צריך לבדוק שמשהו עובד — `TESTS.md`
4. אם רוצים למחוק dashboard ישן — `DELETION-CHECKLIST.md`
